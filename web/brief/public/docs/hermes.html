Hermes Agent

Nous Research 出品的自学习 AI Agent，内置技能学习闭环 + 跨平台消息网关，可通过 DD Tokens 接入任意大模型

概述

Hermes Agent 是 Nous Research 出品的开源 AI Agent，主打「会成长的智能体」——它是目前少数内置学习闭环的 Agent：能从对话中自动生成技能、在使用中持续打磨技能、主动提醒自己沉淀知识，并通过 FTS5 全文索引检索过往会话，跨 session 维护对你的用户画像。Hermes 不依赖本地机器，从 5 美元的 VPS 到 GPU 集群、再到几乎零成本闲置的 Serverless 平台都能跑。

通过对接 DD Tokens，您可以获得：

安装

Linux / macOS / WSL2 / Termux

bash 
curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash

Windows（PowerShell，原生支持仍在 Early Beta）

powershell 
iex (irm https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.ps1)

安装器会自动处理 uv、Python 3.11、Node.js、ripgrep、ffmpeg 以及一份便携 Git Bash。

完成后重载 Shell 即可启动：

bash 
source ~/.bashrc    # 或 source ~/.zshrc
hermes              # 进入终端 UI，开始对话

接入 DD Tokens

Hermes 自带 hermes model 命令，可选 Nous Portal / OpenRouter / OpenAI / 自建端点等。DD Tokens 提供 OpenAI 兼容 API，作为「OpenAI 自定义端点」接入即可一次拿到全模型矩阵。

方式一：通过 hermes model 交互式配置（推荐）

bash 
hermes model        # 进入模型选择向导

向导会依次询问：

步骤 输入
Provider 选 OpenAI（或 Custom OpenAI endpoint）
API Base URL https://api.ddtokens.cn/v1
API Key 你的 DD Tokens 密钥（sk-...）
Model 填入想用的模型 ID，如 gpt-5.4、claude-sonnet-4-6、deepseek-v3.2、gemini-3.1-pro-preview

方式二：通过 hermes config set 直接写入

bash 
hermes config set llm.provider openai
hermes config set llm.base_url https://api.ddtokens.cn/v1
hermes config set llm.api_key sk-你的DD_Tokens密钥
hermes config set llm.model gpt-5.4

写入后用 hermes 启动一次验证连接。

方式三：环境变量（适合 Docker / Serverless 部署）

bash 
export OPENAI_API_BASE=https://api.ddtokens.cn/v1
export OPENAI_API_KEY=sk-你的DD_Tokens密钥
export HERMES_MODEL=gpt-5.4
hermes

切换模型只需 hermes model 或修改 HERMES_MODEL，无需改任何代码。

方式四：Anthropic 原生协议（主用 Claude 强烈推荐）

Hermes 把 Anthropic 作为一等公民 Provider，内部 wire protocol 称为 anthropic_messages，相比走 OpenAI 兼容路径有独家加成：

CLI 配置：

bash 
hermes config set llm.provider anthropic
hermes config set llm.base_url https://api.ddtokens.cn
hermes config set llm.api_key sk-你的DD_Tokens密钥
hermes config set llm.model claude-sonnet-4-6

环境变量等价写法：

bash 
export ANTHROPIC_BASE_URL=https://api.ddtokens.cn
export ANTHROPIC_API_KEY=sk-你的DD_Tokens密钥
export HERMES_MODEL=claude-sonnet-4-6
hermes

base_url 不要带 /v1：必须是 https://api.ddtokens.cn。Anthropic 协议会自动拼接 /v1/messages，写错会变成 .../v1/v1/messages 触发 404。

Hermes 会根据 URL 自动识别协议（路径含 /anthropic 直接走 anthropic_messages）。若使用 LiteLLM 代理等非标准端点，可手动指定：

bash 
hermes config set llm.api_mode anthropic_messages

常用功能速查

从 OpenClaw 迁移

如果你之前使用 OpenClaw，Hermes 内置了一键迁移工具：

bash 
hermes claw migrate              # 交互式完整迁移
hermes claw migrate --dry-run    # 预览将迁移哪些内容
hermes claw migrate --preset user-data   # 仅迁移用户数据，不含密钥
hermes claw migrate --overwrite  # 覆盖冲突

将自动导入 SOUL.md、记忆 (MEMORY.md / USER.md)、用户技能、命令白名单、消息平台配置、API 密钥（Telegram / OpenRouter / OpenAI / Anthropic / ElevenLabs）、TTS 资源、工作目录指令等。
